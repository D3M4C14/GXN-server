* 线程

* 编译运行命令  
  `g++ -g -Wall -std=c++11 -o server 1_server.cpp -lpthread`  
  `./server`

* **1.** 线程基础用法  
  创建线程、线程属性、退出。

* **2.** 线程取消  
  线程取消回调函数、设置取消方式和类型。

* **3.** 线程信号量  
  信号量创建、属性操作等。

* **4.** 线程互斥锁  
  互斥锁创建、属性类型操作、检错锁等。

* **5.** 线程条件变量  
  条件变量创建、属性类型、广播通知、唤醒指定线程等。

* **6.** 多线程创建进程的锁继承  
  pthread_atfork使用。

* **7.** 多线程信号  
  多线程信号发送、屏蔽、处理(可以参考signal项更具体的示例)。

* **8.** 多线程私有变量  
  线程私有变量、线程单次初始化。
