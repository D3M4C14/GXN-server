* 进程

* 编译运行命令  
  `g++ -g -Wall -std=c++11 -o server 1_server.cpp -lpthread -lrt`  
  `./server`  
  其实只有第4例需要 -lrt 由于使用了POSIX共享内存对象

* **1.** 多进程用法  
  fork创建子进程、进程结束信号、注册退出函数、进程管道通信。

* **2.** 信号集  
  多进程同步、sem信号量PV操作。

* **3.** 共享内存  
  多进程通过共享内存交换数据。

* **4.** 共享内存POSIX方法  
  `shm_open` `shm_unlink`等。

* **5.** 共享内存无关联进程一般用法  
  无关联进程间共享内存使用(同时启动多个程序)等。

* **6.** 消息队列用法  
  进程间消息队列传递数据。

* **7.** 进程间传递文件描述符  
  进程间通过辅助数据传递文件描述符。  
  (Ubuntu 20.04正常,Ubuntu 16.04还有点问题)
